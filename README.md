# Introduction
This is my Nvim config for Linux. This is nowhere near finished, I plan to add a lot of stuff to this later on,
alongside changing the overall look. This config is based on the latest nvim stable release.

# Features
- [LazyVim](https://github.com/folke/lazy.nvim) as plugin manager
- [blink.cmp](https://saghen/blink.cmp) for code snippets and auto completion
- [vim-fugitive](https://github.com/tpope/vim-fugitive) for Git integration
- [lualine.nvim](https://github.com/nvim-lualine/lualine.nvim) as the statusline (might change to mini.statusline)
- [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig) for language server protocol (LSP)
- [telescope](https://github.com/nvim-telescope/telescope.nvim) as the fuzzy finder
- [nvim-highlight-colors](https://github.com/brenoprata10/nvim-highlight-colors) for highlighting colors
- [mini.nvim](https://github.com/nvim-mini/mini.nvim) for Lua snippets
- [trouble.nvim](https://github.com/folke/trouble.nvim) for diagnostics and troubleshooting
